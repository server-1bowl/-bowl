# 서버 1인분 스터디

!SOPT 31기 4인의 1인분을 하기 위한 서버 스터디! 

- 각자 좋아하는 음식-bowl 의 형태로 레포를 팝니다!
- 앞으로 이어질 스터디를 해당 레포에 올립니다!
- 각자의 레포의 pr을 통해 스터디 한 내용을 리뷰합니다!

### 스터디

- 시간 : 매주 수요일 오후 5시에
- 장소 : 온 or 오프라인 으로
- 내용 : 세미나 복습 및 추가시도

### 과제

- 방법 : 개인 레포에 해당 주차의 브런치를 파 push 하고 main 브런치로 pr하기
- !후 처리! : 리뷰어에 나머지 3명의 아이디를 추가 후, 리뷰를 요청하기!
